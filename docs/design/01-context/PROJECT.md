# PROJECT.md

## ビジョン

忘却曲線に基づく暗記支援アプリを開発し、暗記に困っている人を救済したい

## 解決する課題

忘却曲線による復習をしたいが、復習のサイクルの失念や怠慢による未実施などで、繰り返すことができず知識の長期定着をすることができない。

## ターゲットユーザー

- 学習者（学生）: 日々の復習で知識を長期定着させたい。
- 社会人学習者: 隙間時間で効率的に復習し、成果を可視化したい。

## 主要機能（MVP）

- 認証
  - 複数のユーザーを特定する必要があるため、認証機能を設ける。
- 学習項目（learning items）のCRUD
- 復習（review）完了操作と次回復習日の更新
- 本日復習が必要な項目の一覧（優先表示）
